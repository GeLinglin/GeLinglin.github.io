# 文人画 梅 兰 竹 菊

  • 当时文人画常见的题材。
  • 文人画是中国古代美术传统中能够重要方面它出现在中国封建社会的后期。
  • 13 世纪后半期南宋灭亡，新朝由异族统治不重视文化取消画院制度，当时的画家有制度的保护在宫廷内的画院作画。
  • 被外族所统治导致画家处于在野，他们借笔墨写仇记恨，表达自己洁身自好的操守和率意而为的人生态度形成了中国文人画的传统。
    倪瓒，元末明初画家、诗人与黄公望、王蒙、吴镇合成‘’元四家‘’。
  • 文人用水墨画表达自己的情绪用以基本的表现手段在宣纸上用墨作画。
  • 梅花 - 傲借助梅花表达自己的高洁
  • 兰花 - 幽情景自守的节操
  • 竹 - 坚 表达虚心静洁挺拔向上
  • 菊花 - 淡自己始终如一的爱国之心

# 题画诗

  • 诗或题字就是文人画的中心思想
  • 文人画出现在中国古代社会后期，至今尚残存影响。大部分文人画由在野知识分子完成，有自我表现的性质，多表达某种个人情怀，消极成分较多。
