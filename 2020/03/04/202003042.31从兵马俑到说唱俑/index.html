# 西安咸阳秦始皇兵马俑博物馆

  • 中国早起的艺术都是从墓室里发现的，秦汉时期帝王大部分都是兵马俑。中国古代秦汉时期好武功善征战早起帝王通过武功打下自己的江山。死亡之后以军阵做陪葬品体现了政治理想军事理想。

# 霍去病

  • 汉族西汉名将、军事家《马踏匈奴》。《马踏飞燕》位于甘肃武威县台东汉墓。东汉墓室军人战马变成说唱艺人的作品。
  • 墓葬艺术反应墓主信仰和生活。早期君主保持战争习惯，以兵马俑为主要陪葬无。后期君主耽于享乐，以娱乐生活所需要的物品为主要陪葬物。
