# 简要

  • 通过降级实现设备远程登录，后首次刷机需拆外科，通过 usb 转 ttl 串口模块连接主板再与电脑链接通过 putty 打开串口会话，修改
    /etc/rc.local，传入 breed，再用 mtd 写入固件。在给供电前需要按住重置键按到所有指示灯齐亮一下即可松开。进行设备后台，选择固件更新，上传固件包，设备重启，刷入完成。

# 注意项

  • 使用降级软件时会更改当前静态地址，用完后手动恢复。
  • 螺丝是内三角螺丝最好找合适的工具拧，以免螺丝拧花了
  • 修改 /etc/rc.local 文件注意在 #号框上下中间添加
    /etc/init.d/dropbear enable
    /etc/init.d/dropbear start
  • 在进行串口连接时，连接线需要公母杜邦线，双母线可以在一头加一针脚之类的，按参考网址的方法，找个合适的位置抵住即可
  • 串口连接（RX 与 TX 交叉连接，GND 直连）
  • 刷完 breed 不死后台，按重置键的时候注意指示灯齐亮即可
