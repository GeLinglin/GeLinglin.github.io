# 简要

  • 主板取出确认主板版本信息，找到短接点，打开烧录工具，连接 HDMI 口旁的 usb 口，保持主板短接并进行供电，指示灯变色，软件显示设备成功，刷入底包。找空 U 盘刷入固件包，将 U 盘插入网口旁的 usb 口，再次给设备供电，待指示灯无变化，刷入完成。

# 注意项

  • 首次刷机需取出主板进行操作

# USB_Burning_Tools 烧录时

+ 用双公头 usb 一端电脑，另一端需要插入靠近 hdmi 口旁的 usb 口。首次刷机要先在主板上在短接两点保持短接并同时同上电源，具体短接点位置可查看此处链接，可以先观察主板指示灯确定情况，再查看软件是否已经显示设备成功连接上，然后可以不用继续进行短接主板也不需要继续供电。

# balenaEtcher 刷入时

  • 烧录完底层包完后，找一个比固件包文件大点的 U 盘（刷入时会将 U 盘格式化注意转存备份），再选择包进行刷入。
  • U 盘刷入完后，注意插入靠近网络端口旁的 usb，然后接上电源，在期间指示灯会不断变色熄灭。
  • 在指示灯没反应时接上网线，通过路由后台查看设备或是探测网段在线主机查找其 IP 地址。
